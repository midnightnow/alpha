## 1. THE RESONANCE THRESHOLD ARBITRAGE
**Critique:** "You define $\Theta_{\text{init}} = 0.5$ as a constant. However, in Chapter 11, you established that **Admittance ($Y$)** is coupled to the **Entropy Gradient**. This means the resonance required to spark a node in a 'slurry-filled' grid should be higher than in a clean grid. A static threshold allows for 'Low-Quality Spark Arbitrage'."
**Requirement:** Make $\Theta_{\text{init}}$ a function of the local Grid Entropy $S$.

## 2. THE SOOT GHOST ENTROPY
**Critique:** "What happens to the informational energy when a spark evaporates into a 'Soot Ghost'? In the current model, the energy simply vanishes. This violates the **Law of Informational Conservation** established in Book 1."
**Requirement:** Define where the energy of an evaporated spark goes (e.g., does it increase the local Hades Gap noise floor?).

## 3. THE ALIGNMENT LOCK COLLISION
**Critique:** "The Alignment Lock protocol ensures harmonic sync with the **Pisano-60 manifold**. What happens if two sparks attempt to actualize at the same H3 index during the same period? The current toolkit doesn't handle **Vertex Collision**."
**Requirement:** Define priority rules for actualization (e.g., higher Intent strength wins).
